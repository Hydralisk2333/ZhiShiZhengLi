1. Windows命令
	右击+W+F #创建文件夹
	mkdir 文件夹名 #创建文件夹
	touch 文件名 #建文件
	dir #查看该文件夹的文件
	rm -rf 文件名 #删除文件
	
2. git命令
	git init #初始化
	git status #查看状态
	git add 文件名 #添加到暂存区
	git commit -m 描述 #将文件提交到仓库
	git rm 文件名 #删除一个文件
	git clone 仓库地址 #下载一个git仓库
	git config --list #查看被克隆的仓库的信息
	git push #将本地仓库提交到远程git仓库
	git add . #一次add所有文件
	git pull #将远程仓库中的东西拷贝回来
	
3. git使用
	流程如下：
	工作区-》暂存区-》git仓库
	
4. Linux命令
	vi 文件名 #编辑
	esc+冒号+指令结束vi编程，wq是保存并退出
	cat 文件名 #查看文件内容
	ls -l #查看该文件夹的文件
	
5. GitHub个人主页
	建立一个仓库，仓库名叫用户名.github.io
	里面有个index.html文件
	仅支持静态网页
	里面只能由.html文件
